# Video Intérprete Para Personas con Discapacidad Auditiva
### Aplicación de Visión Artificial para la detección de letras del alfabeto del Lenguaje de Señas Mexicano (LSM).

## 🚀 Características
- Detección precisa de las señas con Mediapipe Holistic
- Lectura de Texto a Voz por medio de la libreria pyttsx3
- Ignora personas y señas de fondo de acuerdo a la distancia que se encuentren de la camara.
- Interfaz amigable para visualizar la letra con mayor probabilidad de ser detectada
